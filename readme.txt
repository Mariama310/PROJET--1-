Bienvenue dans le fichier README pour l'installation des dépendances de votre application Python.

Pour exécuter votre application, certaines bibliothèques Python externes doivent être installées. Nous avons préparé un script d'installation rapide pour vous aider à configurer votre environnement avec toutes les bibliothèques nécessaires.

------------------------------------
INSTRUCTIONS D'INSTALLATION POUR WINDOWS
------------------------------------

1. Double-cliquez sur 'install_dependencies.bat' depuis votre explorateur de fichiers pour lancer le script.
2. Une fenêtre d'invite de commande s'ouvrira et commencera l'installation des bibliothèques.
3. Une fois l'installation terminée, vous pourrez fermer l'invite de commande et exécuter votre application Python.

Remarque: Si vous rencontrez des erreurs liées à l'autorisation, faites un clic droit sur 'install_dependencies.bat' et choisissez 'Exécuter en tant qu'administrateur'.

------------------------------------
INSTRUCTIONS D'INSTALLATION POUR LINUX/MAC
------------------------------------

1. Ouvrez le terminal.
2. Changez le répertoire courant du terminal pour aller là où est le script. Par exemple, si le script est dans le dossier Téléchargements, tapez:
   cd ~/PROJET--1-
3. Rendez le script exécutable avec la commande suivante :
   chmod +x install_dependencies.sh
4. Exécutez le script en tapant :
   ./install_dependencies.sh
5. Le terminal effectuera l'installation des bibliothèques nécessaires.
6. Une fois l'installation terminée, vous pouvez fermer le terminal et exécuter votre application Python.

Remarque : Sur certains systèmes, vous devrez peut-être utiliser 'pip3' au lieu de 'pip'. Le script fourni suppose que 'pip3' est utilisé pour les systèmes Linux/Mac.

Si vous rencontrez des problèmes ou avez besoin d'aide supplémentaire, n'hésitez pas à contacter un des membres de l'équipe ou à consulter la documentation officielle de Python pour plus d'informations sur l'installation des paquets avec pip.
